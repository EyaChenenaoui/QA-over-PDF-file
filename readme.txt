
This project aims to provide answers to questions posed by the user regarding a loaded PDF file. 
The backend utilizes Retrieval-Augmented Generation (RAG) principles, ensuring accurate and relevant responses. 
For the user interface, Streamlit has been employed to create an intuitive and user-friendly experience.
